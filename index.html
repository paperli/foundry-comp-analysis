At a glance — Astrocade vs Tesana vs Foundry

One-glance summary of where the two AI prompt-to-game platforms sit, and the lane Weekend / Foundry owns.

Dimension Astrocade Tesana Foundry (Weekend) · us
Output & ownership 2D casual mini-games. No export — platform lock-in. 2D + 3D games. Real Godot project; export to macOS / Win / Web / source — you own it. TV games for the living room; owned + licensed show formats.
Device / platform Phone-first: browser + iOS / Android. Browser + iOS / Android; desktop / web export. TV-first; phones connect as controllers.
Play context Solo; async social feed. Solo; async community feed. Co-present, in-the-room: couch local multiplayer (game night) + single player.
Input / interaction Touch + text prompt. Mouse / touch / keyboard; voice only to prompt. Voice-first + touch, gesture, gyro, motion.
IP / content Original user games. Original user games. Major TV-show IP — Jeopardy, Wheel of Fortune, Family Duel, Are You Smarter…
Creation UX Wish → named concept → agent team (~10 min). Prompt / template → live agent build (~3 min); real-time thought-stream. Describe a game-night idea; tuned for shared TV play.
Creator economy & scale Play payouts + $10M fund, 100K-play gate. $56M (Sea, Sequoia); ~5M MAU. Play + social-clip payouts (per 1K views), weekly via Stripe. Funding / MAU undisclosed; ~10K paying users. —
